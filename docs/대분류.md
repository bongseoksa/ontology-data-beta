# AI 기반 프로젝트 가격산정 시스템 설계 계획서

---

# 1. 프로젝트 목표 정의 (Project Goal Definition)

## 1.1 문제 정의
중소기업 대표가 업로드한 사업계획서를 분석하여  
프로젝트 구조를 도출하고, 최종 가격을 자동 산정하는 시스템을 구축한다.

## 1.2 핵심 목표
- 사업계획서 → 구조화된 프로젝트 모델 변환
- 구조화된 프로젝트 모델 → 원가 계산
- 원가 → 전략적 가격 산정
- 전문가 시뮬레이션 기반 다중 판단 구조 구현

## 1.3 최종 산출물
- 프로젝트 가격 산정 결과
- 가격 산정 근거 리포트
- 리스크 및 전략 가중치 설명

---

# 2. 도메인 범위 정의 (Domain Boundary)

## 2.1 산업 범위
- 소프트웨어 산업 한정

## 2.2 계약 유형 범위
- Project-Based 계약
- Bid-Based 계약
- Government 과제 (2단계 확장)

## 2.3 제외 범위
- 순수 SaaS 구독 가격 전략
- 제조/건설 산업

---

# 3. 전체 시스템 구조 개요

사업계획서

}

# 6. Multi-Agent 전문가 구조 설계

}

## 6.2 사업 전문가 (Business Expert)

### 출력

  base_cost:

  recommended_margin:
}

---

# 7. 가격 계산 엔진 설계

## 7.1 기본 공식

Base Cost =
(Resource × Duration × UnitCost)
+ Overhead
+ ExternalCost

Final Price =
Base Cost
× (1 + Margin)
× (1 + RiskPremium)
× StrategicAdjustment

# 8. RAG 설계

## 8.1 목적
- 유사 프로젝트 사례 검색
- 산업 평균 단가 참고
- 공공 단가 기준 반영
- 리스크 사례 보강

## 8.2 사용 영역
- 기술 전문가 판단 근거 보강
- 사업 전문가 시장 분석 보강
- 경제 전문가 단가 기준 보강

# 최종 정의

이 시스템은 단순 견적 자동화가 아니라

의사결정 자동화 시스템이다.
