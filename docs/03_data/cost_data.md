## 2. SW사업 대가산정 가이드

### 2.1 공식 가이드

| 문서명 | 발행처 | 최신 버전 | 적용 범위 |
|--------|--------|----------|----------|
| **SW사업 대가산정 가이드** | 과기정통부/SW산업협회 | 2024년판 | 공공 SW사업 |
| **SW개발비 대가기준** | 행정안전부 | 고시 | 정부 발주 |
| **기능점수(FP) 산정 가이드** | KOSA | 연간 | FP 방식 적용 시 |

### 2.2 대가산정 공식 구조

```
[기능점수 방식]
SW개발비 = 기능점수 × 기능점수당 단가 × 보정계수

[투입공수 방식]
SW개발비 = (직접인건비 + 제경비 + 기술료) + 직접경비
         = 직접인건비 × (1 + 제경비율 + 기술료율) + 직접경비
```

### 2.3 보정계수 항목

| 보정계수 | 범위 | 적용 조건 |
|----------|------|----------|
| 규모 보정 | 0.65 ~ 1.28 | FP 규모에 따라 |
| 애플리케이션 복잡도 | 0.9 ~ 1.1 | 시스템 복잡도 |
| 언어 보정 | 0.9 ~ 1.2 | 개발 언어별 |
| 품질/특성 보정 | 0.95 ~ 1.05 | 요구 품질 수준 |
