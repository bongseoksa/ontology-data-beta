# Phase 3-3: 가격/시장 데이터 소스

---

## 1. 공공 입찰 데이터

### 1.1 나라장터 (조달청)

| 데이터 유형 | 설명 | 접근 방법 |
|------------|------|----------|
| **낙찰 정보** | 입찰 결과, 낙찰 금액 | g2b.go.kr (공개) |
| **예정가격** | 발주 기관의 예산 | 입찰 공고 |
| **투찰 현황** | 경쟁 업체 수, 투찰률 | 개찰 결과 |
| **계약 정보** | 최종 계약 금액, 조건 | 계약 공개 |

### 1.2 수집 가능한 데이터 항목

```
[입찰 공고 데이터]
├── 공고번호
├── 사업명
├── 발주기관
├── 추정가격/예정가격
├── 입찰 방식 (최저가, 협상 등)
├── 납품 기한
└── 참가 자격 조건

[낙찰 결과 데이터]
├── 낙찰자 정보
├── 낙찰 금액
├── 낙찰률 (낙찰가/예정가)
├── 참여 업체 수
└── 유찰 여부
```

### 1.3 데이터 활용 방법

| 분석 목적 | 활용 데이터 | 산출물 |
|----------|------------|--------|
| 적정 가격 추정 | 유사 사업 낙찰가 | 시장 가격 범위 |
| 경쟁 강도 분석 | 투찰 업체 수 | 경쟁 지수 |
| 낙찰률 분석 | 낙찰가/예정가 비율 | 할인율 기준 |

## 2. 민간 SI 계약 데이터

### 2.1 데이터 소스

| 소스 | 설명 | 접근성 |
|------|------|--------|
| **IT서비스 기업 공시** | 상장사 계약 공시 | DART (공개) |

| **채용 공고** | 연봉 정보로 단가 추정 | 잡코리아, 원티드 등 |

### 2.2 상장 IT기업 계약 공시 분석

```
[공시 항목]
├── 계약 상대방
├── 계약 금액
├── 계약 기간
├── 계약 내용 (대분류)
└── 매출 비중

[분석 가능 정보]
├── 프로젝트 규모별 단가
├── 기간 대비 금액 비율

```

| 주니어 개발자 | 400~600만원 | 위시켓 |
| 디자이너 | 400~800만원 | 크몽, 라우드소싱 |

## 3. 마진율 벤치마크

### 3.1 상장사 재무제표 기반 (DART)

| 지표 | 데이터 소스 | 활용 |
|------|------------|------|
| **매출총이익률** | 손익계산서 | 프로젝트 마진 추정 |
| **영업이익률** | 손익계산서 | 전체 사업 수익성 |
| **원가율** | 손익계산서 | 원가 비중 분석 |

### 3.2 IT서비스 업계 평균 마진율

| 구분 | 매출총이익률 | 영업이익률 |
|------|:----------:|:--------:|
| 대형 SI (삼성SDS 등) | 15~20% | 5~10% |
| 중형 SI | 20~30% | 8~15% |
| 솔루션 기업 | 40~60% | 15~25% |
| 에이전시 | 30~50% | 10~20% |

### 3.3 마진 구조 분석

```
[프로젝트 원가 구조 (일반적)]
├── 직접 인건비: 50~60%
├── 간접비/제경비: 20~30%
├── 외주/구매비: 5~15%
└── 영업이익: 10~20%

[마진 조정 요소]
├── 전략적 중요도: ±5%
├── 후속 사업 가능성: ±5%
├── 경쟁 강도: ±10%
└── 리스크 수준: ±5%
```

```

└── 경쟁 심화
```

## 5. 경쟁사 가격 정보

### 5.1 가격 정보 수집 방법

| 방법 | 설명 | 신뢰도 |
|------|------|:------:|
| 공개 입찰 결과 | 나라장터 낙찰 정보 | 높음 |
| RFP 응찰 경험 | 자사 입찰 경험 축적 | 높음 |
| 업계 네트워크 | 비공식 정보 | 중간 |
| 고객 피드백 | 경쟁 견적 비교 정보 | 중간 |

### 5.2 경쟁 분석 프레임워크

```
[경쟁 강도 평가]
├── 참여 업체 수: 3~5개 (보통), 6+ (높음)
├── 가격 민감도: 공공(높음), 민간(중간)
├── 기술 차별화: 있음(낮음), 없음(높음)
└── 기존 관계: 있음(낮음), 없음(높음)

[경쟁 대응 전략]
├── 가격 경쟁: 원가 절감, 마진 조정
├── 가치 경쟁: 부가가치 강조
└── 관계 경쟁: 신뢰 구축, 실적 활용
```

## 6. 가격 협상 범위 데이터

### 6.1 협상 가능 범위 (경험치 기반)

| 계약 유형 | 초기 제시가 대비 | 비고 |
|----------|:---------------:|------|
| 공공 입찰 | -5~10% | 협상 제한적 |
| 민간 수의계약 | -10~20% | 협상 여지 있음 |
| 장기 계약 | -15~25% | 물량 할인 |
| 전략 고객 | -20~30% | 관계 투자 |

### 6.2 할인 조건

```
[할인 적용 조건]
├── 선금 지급: 3~5% 할인
├── 장기 계약 (2년+): 10~15% 할인
├── 반복 거래: 5~10% 할인
├── 레퍼런스 허용: 5% 할인
└── 마케팅 협조: 3~5% 할인

[추가 과금 조건]
├── 촉박한 일정: +10~20%
├── 불명확한 요구사항: +10~15% (리스크)
├── 특수 보안 요구: +5~10%
└── 야간/휴일 작업: +15~25%
```
